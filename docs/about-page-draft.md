## About

Shoshin is twelve tools for twelve weeks.

Each tool asks something of your body or your time. One asks you to breathe. One asks you to sit in silence. One asks you to write something down and watch it disappear. One asks you to do an ordinary thing — wash a dish, fold a shirt — and return.

None of them teach. None of them explain. None of them congratulate you for showing up.

The tools are drawn from the Zen tradition, particularly Shunryu Suzuki's *Zen Mind, Beginner's Mind*. They are not a summary of that book. They are small rooms you can walk into, built from the same material.

---

Nothing leaves your device. There is no account, no profile, no dashboard. Your progress is stored in your browser and nowhere else. If you clear it, it is gone.

There is no tracking. No analytics. No encouragement. No streak counter. No notification asking where you've been.

You have a key. The key opens the door. That is the entire relationship.

---

Shoshin was built with static HTML, CSS, and vanilla JavaScript. No framework. No build step. Sound is generated live through the Web Audio API — singing bowls, breathing tones, water. The calligraphy tool uses canvas. The whole thing is hosted on Cloudflare Pages and weighs almost nothing.

This is deliberate. The tool should be so simple that you forget it is there. The practice is what remains.
